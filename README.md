# 왔다감

자신의 위치에 메모를 남기는 웹 서비스

## 기획 배경

관광지나 유적지를 둘러보면 여행객들이 남긴 낙서들을 볼 수 있습니다. 각자의 감상을 공개된 장소에 남긴다는 점이 재미있다고 느껴져서 이를 온라인으로 옮겨보려고 합니다. 낙서들 중 'XXX 왔다감' 이라는 문구가 기억에 남아서 이름을 왔다감으로 정했습니다.

## 서비스 소개

온라인 방명록 서비스입니다.  지도 상의 자신의 현재 위치에 메모를 남길 수 있습니다. 주변에 작성된 메모들을 확인 할 수 있고, 비밀번호를 알고 있다면 수정/삭제를 할 수 있습니다.

## 주요 기능

- 현재 위치에 메모 작성
- 현재 위치 주변 메모 조회
- 비밀번호를 입력해서 메모 수정/삭제
